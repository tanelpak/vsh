## Apache, PHP, MySQL
Esmalt, et alustada tööd, paigaldame "apache" masinale apache, MySQLi ja PHP.  
Apache paigaldamine :  
```
sudo apt-get update  
sudo apt-get install apache2  
```

```
sudo apt-get install mysql-server  
mysql_secure_installation
```
